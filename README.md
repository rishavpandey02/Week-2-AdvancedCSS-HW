# Week-2-AdvancedCSS-HW

This project consists of the creation of my portfolio, with the use of HTML of CSS. 

## Features

- All application links function correctly.
- CSS selectors and properties have been consolidated and organized to follow a semantic structure.
- The CSS file is properly commented for enhanced code understanding.

## Installation

To run this project, follow the steps below:

1. Download the project files to your local machine.
2. Open the HTML file (`index.html`) in a web browser.

Alternatively, you can also access the webpage by visiting the live version (https://rishavpandey02.github.io/Week-2-AdvancedCSS-HW/).

## Usage

Once the webpage is loaded, you can navigate through different sections using the links in the header:

- "About Me" section provides information about the author.
- "Work" section displays project images.
- "Contact Me" section includes contact information and links to external profiles.

Feel free to explore the webpage and interact with the provided links.

## File Structure

The project files are organized as follows:

- `index.html`: The main HTML file containing the structure and content of the webpage.
- `Assets/CSS/reset.css`: CSS file responsible for resetting default browser styles.
- `Assets/CSS/style.css`: CSS file containing the modified styles for the webpage.
- `Assets/image/`: Folder containing image files used in the project.
